Build a Full Stack Learning Management System (LMS) web application as a working MVP.

The application should support Students, Instructors, and Admin roles with authentication and basic learning features.

🧠 PROJECT GOAL

Create a simple but functional LMS where:

Instructors can create courses and upload lessons
Students can enroll, learn, take quizzes, and track progress
Admin can manage users and courses

Keep the project simple, clean, and fully runnable in Replit.

⚙️ TECH STACK
Frontend:
React (Vite)
React Router DOM
Axios
Basic CSS (no heavy UI libraries)
Backend:
Node.js
Express.js
JWT Authentication
bcrypt for password hashing
Database:
PostgreSQL
Optional Integrations (simple/mock if needed):
Cloudinary (for video/image URLs)
Razorpay (mock payment or test mode only)
